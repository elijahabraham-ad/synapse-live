# Client logos

Drop client/company logos here. Reference them from `TrustedBySection`'s
`companies` prop on the homepage as `/clients/your-logo.svg`.

Recommended:
- Format: SVG (preferred) or transparent PNG
- Height: 80px (renders at max 40px, so 2x for retina)
- Background: transparent
- Color: single-tone works best with the grayscale-to-color hover effect
